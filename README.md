# Count1By1Texteditor

## URL
https://count1by1texteditor.netlify.app/

## 機能
- 入力した文字数を教えてくれるテキストメモアプリです。
- ワンクリックでクリップボードコピーとtxtファイルとして保存も可能です(保存先はダウンロードフォルダ)
